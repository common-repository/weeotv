A plug-in for turning WordPress in a Semantic Web Tv.

== Description ==

This plug-in turns WordPress in a Semantic Web Tv, by allowing editors to manually (or automatically) ingest
Video contents and related data in a schema.org compatible format.

The plug-in does not provide a front-end.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `weeotv.zip` to the `/wp-content/plugins/` directory
2. Unzip it
3. Activate it through the 'Plugins' menu in WordPress
